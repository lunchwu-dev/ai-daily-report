# AI日报生成系统 v1.0

## 项目说明
自动生成全球AI大事件热点日报，包含中英文双语摘要。

## 版本信息
- **版本**: v1.0
- **更新日期**: 2026-03-02
- **更新内容**: 完整执行要求规范、响应式HTML模板、自动化部署流程

## 完整执行要求

### 1. 执行时间
**每天 7:30 AM**（Asia/Shanghai 时区）

### 2. 执行步骤

| 步骤 | 任务 | 详细要求 |
|:---|:---|:---|
| **1** | **收集AI热点** | 国内（36氪、机器之心、量子位、智东西）+ 海外（TechCrunch、The Verge、OpenAI Blog、Google AI Blog） |
| **2** | **获取GitHub模板** | 仓库：`lunchwu-dev/ai-daily-report`，文件：`template.html` |
| **3** | **生成HTML报告** | 使用模板样式，填充今日热点，每个事件包含真实来源链接 |

| **5** | **Cloudflare部署** | 主报告（`ai-daily-report`）+ 历史版本（`ai-report-YYYY-MM-DD`）+ 索引页面（`ai-daily-report-index`） |
| **6** | **QA验证** | 检查链接可访问性、页面功能完整性，**验证通过后才能发送通知** |
| **7** | **发送邮件** | 收件人：lunchwu@gmail.com，使用应用专用密码 |
| **8** | **发送飞书消息** | 目标：user:ou_11ad053de00f1b52a9bf7fd6f0dc6a34 |

### 4. 质量要求

#### 来源链接
- 每个"查看详情"必须是真实URL
- 严禁使用 `href="#"`
- 优先官方来源，其次权威媒体

#### 双语摘要
- 中英文对应同一批Top 3热点
- 英文翻译地道专业

#### 响应式设计
- PC端：多栏布局
- 移动端：单列布局

#### 链接完整性（v1.0新增）
- [ ] 报告主链接公网可访问（HTTP 200）
- [ ] 历史报告查询链接可点击跳转
- [ ] 每个数据来源的"查看详情"链接可访问
- [ ] 索引页面"查看更多"链接正常
- [ ] 所有外部链接使用 `target="_blank"` 在新窗口打开

**验证方法：**
- 使用curl检查HTTP状态码
- 至少抽查3个链接验证可访问性
- 确保无 `href="#"` 占位符
