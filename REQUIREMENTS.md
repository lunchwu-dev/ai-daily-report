# AI日报生成系统 v1.1

## 项目说明
自动生成全球AI大事件热点日报，包含中英文双语摘要。

## 版本信息
- **版本**: v1.1
- **更新日期**: 2026-03-03
- **更新内容**: 
  - 新增"今日头条"板块（金色高亮，四层阅读层级）
  - 替换"市场情绪"为"AI概念股分析"（A股+美股）
  - 优化往期报告展示逻辑（首份报告自动隐藏）
  - 新增响应式导航（PC顶部+移动端底部）

## 完整执行要求

### 1. 执行时间
**每天 7:30 AM**（Asia/Shanghai 时区）

---

### 2. 执行步骤

| 步骤 | 任务 | 详细要求 |
|:---|:---|:---|
| **1** | **收集AI热点** | 国内（36氪、机器之心、量子位、智东西）+ 海外（TechCrunch、The Verge、OpenAI Blog、Google AI Blog） |
| **2** | **生成今日头条** | Top 3热点，金色高亮，四层阅读层级（摘要→详情→影响→来源） |
| **3** | **生成HTML报告** | 使用template.html样式，填充今日热点，每个事件包含真实来源链接 |
| **4** | **生成JSON摘要** | 文件：`ai-report-YYYY-MM-DD-summary.json`，Top 3热点，中英文对应 |
| **5** | **Cloudflare部署** | 主报告（`ai-daily-report`）+ 历史版本（`ai-report-YYYY-MM-DD`）+ 索引页面（`ai-daily-report-index`） |
| **6** | **QA验证** | 检查链接可访问性、页面功能完整性，**验证通过后才能发送通知** |
| **7** | **发送邮件** | 收件人：lunchwu@gmail.com，使用应用专用密码 |
| **8** | **发送飞书消息** | 目标：user:ou_11ad053de00f1b52a9bf7fd6f0dc6a34 |

### 3. 报告结构（v1.1）

1. **今日头条**（金色高亮边框）- Top 3热点，四层阅读层级
2. **国内热点事件**（红色边框）
3. **海外热点事件**（蓝色边框）
4. **AI概念股分析**（绿色边框）- A股+美股双板块
5. **业界领袖观点**（紫色边框）
6. **往期报告导航**（黄色边框）- 非首份报告时显示

---

### 4. 质量要求

#### 来源链接
- 每个"查看详情"必须是真实URL
- 严禁使用 `href="#"`
- 优先官方来源，其次权威媒体

#### 双语摘要
- 中英文对应同一批Top 3热点
- 英文翻译地道专业

#### 响应式设计
- PC端：多栏布局，顶部固定导航
- 移动端：单列布局，底部固定导航

#### 链接完整性
- [ ] 报告主链接公网可访问（HTTP 200）
- [ ] 历史报告查询链接可点击跳转
- [ ] 每个数据来源的"查看详情"链接可访问
- [ ] 所有外部链接使用 `target="_blank"` 在新窗口打开

---

### 5. 检查清单（必须执行）

- [ ] HTML文件大小 > 10KB
- [ ] JSON摘要文件存在且格式正确
- [ ] 今日头条板块金色高亮显示
- [ ] AI概念股板块包含A股和美股
- [ ] 所有"查看详情"链接不是 `href="#"`
- [ ] 往期导航链接使用完整URL（非首份报告）
- [ ] Cloudflare部署成功（HTTP 200）
- [ ] 邮件发送成功
- [ ] 飞书消息发送成功
- [ ] **链接完整性检查通过**
