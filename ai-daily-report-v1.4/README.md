# AI日报 v1.4 版本要求

## 版本信息
- **版本号**: v1.4
- **创建时间**: 2026年3月4日
- **更新人**: AI助手

---

## 核心要求

### 1. 数据时效性要求

**所有数据必须使用截至当日早晨6:30（Asia/Shanghai）的最新数据**

包括但不限于：
- **股票数据**
  - A股：使用前一日收盘数据（当日早晨6:30前最新收盘数据）
  - 美股：使用当日凌晨收盘数据（北京时间早晨6:30前最新数据）
  
- **新闻资讯**
  - 所有新闻必须使用当日早晨6:30前发布的最新内容
  
- **公司动态**
  - 融资信息、产品发布、财报等必须使用最新数据

### 2. 内容去重要求

**各版块之间话题及内容不得出现重复**

- 今日头条只保留最重要的一条新闻详细展开
- 国内/海外热点不再重复今日头条的内容
- 股票涨跌原因与新闻热点不重复描述

### 3. 版块结构

```
AI日报 v1.4
├── 🔥 今日头条（1条最重要新闻）
├── 🇨🇳 国内AI热点（不重复今日头条）
├── 🌍 海外AI热点（不重复上述内容）
├── 📈 AI概念股分析（独立分析）
└── 💡 资产配置与投资建议（综合判断）
```
