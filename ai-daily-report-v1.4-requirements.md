# AI日报 v1.4 版本要求

## 版本信息
- **版本号**: v1.4
- **创建时间**: 2026年3月4日
- **更新人**: AI助手

---

## 核心要求

### 1. 数据时效性要求

**所有数据必须使用截至当日早晨6:30（Asia/Shanghai）的最新数据**

包括但不限于：
- **股票数据**
  - A股：使用前一日收盘数据（当日早晨6:30前最新收盘数据）
  - 美股：使用当日凌晨收盘数据（北京时间早晨6:30前最新数据）
  - 港股：使用前一日收盘数据
  
- **新闻资讯**
  - 所有新闻必须使用当日早晨6:30前发布的最新内容
  - 优先使用权威来源（TechCrunch、36氪、Bloomberg等）
  
- **公司动态**
  - 融资信息、产品发布、财报等必须使用最新数据
  
- **市场数据**
  - ETF涨跌幅、板块表现、指数数据等

### 2. 内容去重要求

**各版块之间话题及内容不得出现重复**

具体规范：
- **今日头条** vs **国内/海外热点**
  - 今日头条只保留最重要的一条新闻详细展开
  - 国内/海外热点不再重复今日头条的内容
  
- **国内热点** vs **海外热点**
  - 严格区分，同一公司的新闻不重复出现
  - 如OpenAI融资在今日头条详述，海外热点中不再重复
  
- **AI概念股分析**
  - 股票涨跌原因与新闻热点不重复描述
  - 聚焦财务数据、技术面、基本面分析
  
- **投资建议**
  - 基于前面所有内容的综合判断
  - 不重复引用具体新闻事件

### 3. 版块结构

```
AI日报 v1.4
├── 🔥 今日头条（1条最重要新闻，详细展开）
├── 🇨🇳 国内AI热点（3-5条，不重复今日头条）
├── 🌍 海外AI热点（3-5条，不重复今日头条）
├── 🛍️ 零售AI应用实践（可选，不重复前面内容）
├── 📈 AI概念股分析（股票数据+独立分析）
└── 💡 资产配置与投资建议（综合判断，不重复）
```

### 4. 数据来源标注

每条新闻/数据必须标注：
- 来源媒体
- 发布时间（需符合6:30前要求）
- 原文链接

### 5. 股票数据格式

**A股示例**:
```
科大讯飞(002230): 52.15元, -3.99%
前一交易日收盘价54.32元，成交额34.68亿元
来源：新浪财经 · 2026年3月3日
```

**美股示例**:
```
NVIDIA(NVDA): +2.93%
美股3月3日（北京时间3月4日凌晨）收盘数据
来源：Bloomberg · 2026年3月3日美股收盘
```
